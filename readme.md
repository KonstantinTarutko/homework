**Программа, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа**

*Алгоритм*

1. Ввод массива
2. Проверка (является ли меньше трех)
3. Создаем новый массив с учетом условий
4. Выводим результат


*Использование программы*

1. Зайти в папку "homework"
2. Выбрать файл "Program.cs"
3. Ввести "dotnet run"
4. Ввести массив
